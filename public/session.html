Session

EndpointDescription
POST /users/sign_inStart a session
DELETE /users/sign_outEnd a session

POST /users/sign_in

Parameters
NameRequired?TypeDescription
emailYesStringEmail for the user.
passwordYesStringPassword for the user.
Example Request
curl -H "Content-Type: application/json" -d '{"email": "user1@example.org", "password": "password"}' -X POST "http://recruiting-api.nextcapital.com/users/sign_in"
Example Response
{    "api_token": "2FxozzbPWxzDTedCrixy",    "email": "user1@example.org",    "id": 1,    "todos": [1,2,3]}

DELETE /users/sign_out

Parameters
NameRequired?TypeDescription
user_idYesStringID for the user.
api_tokenYesStringAPI token for this user's session.
Example Request
curl -H "Content-Type: application/json" -d '{"api_token": "some_token", "user_id": 1}' -X DELETE "http://recruiting-api.nextcapital.com/users/sign_out"
Example Response

Empty on success
